===========================
Quels caractères utiliser ?
===========================

Les \objs \maths, y compris les types "maison", se nomment en respectant les règles suivantes.

	1) \Ttes les lettres \ascii sont utilisables.

	1) Excepté pour les types, les caractères spéciaux suivants sont disponibles.

		* La contre-oblique ''\'' peut être utilisée en toute \1ere position uniquement.

		* Le \_fr ''_'' peut être employé sans aucune restriction, y compris seul.


info::
	''_'' est réservé pour le "pattern matching" via ''match ... with ...'' pour gérer un \ens de cas restants.


caution::
	Les conventions suivantes sont à respecter avec rigueur.

		1) Un nom de la forme ''_...'' signale un \obj n'est pas destiné à un usage publique.

		1) Un nom de la forme ''__...__'' signale un \obj "dangereux".
